Gitstructions

Steps to a Git-filled life

Git Clone

Copying someone else's repository so you can make changes

Clone is used when you want to copy someone else's repository and start making changes / contributions / and additions. It's the first stage to begin working on a project and making contributions. 

          git clone potato
          cd potato
          git pull

Git Pull

Pull someone else's changes into your repo

Pull is used for updating your local copy of the repository when team members make changes. 

          git clone potato
          cd potato
          git pull

Git Push

Push your changes to the main repository

After you make changes, you push them to the main storage center for your code -- the main repository. 

          git clone potato
          cd potato
          vim foo.txt #make changes
          git push
